# Ender Flynnies
Welcome to Ender Flynnies!
Ender Flynnies is a puzzle game about a cat named Flynn that can teleport.
# How to play:
Use the arrow keys to move the pearl around
Spacebar teleports Flynn to the ender pearl if the space is valid
When you come across an item that you can interact with (Lever and other stuff in the future), press Enter to interact
